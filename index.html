Why it's faster?

Web pages require to load lots of resources to render. When a visitor comes to your web site for the first time, their browser will need to load all the content required to render the page. Including a number of CSS files. It takes time.
It takes TOO MUCH TIME.  

Instaload analyses web page of your site and creates special style-code. It enables fast-render of the first screen of web pages. Style-code is a very small piece of CSS code which loads extremely fast. 

So, your every customer instantly sees a web page on your site. Other page code and resources are loading in the background.

How much faster?

By a lot. In some cases, 2s versus 15s!
